# 懂心 dongxin

`dongxin` 是「懂心 - 消灭不开心」项目仓库。

当前仓库采用 monorepo 结构：

- `dongxin_flutter/`: Flutter 客户端
- `dongxin_server/`: 服务端与管理后台骨架

## 当前版本

当前产品阶段为 `0.1`，客户端打包版本为 `2.0.0+201`。

这一版的重点是先把产品主体验做出来，属于**单机版 / 本地版客户端**：

- 客户端可以独立运行
- 不依赖 `dongxin_server` 才能使用
- 声音、MV、视频、宇宙站点等内容当前仍以内置配置为主
- 服务端目录已初始化，但**后台管理、七牛上传、内容接口**还在后续开发中

## 0.1 版本已实现功能

### 1. 发射首页

- 品牌首页与发射入口
- 文案轮播 / 换一句
- 文本输入发射
- 按住说话语音输入
- 语音识别结果实时回填输入框

#### 当前语音实现

- Android：豆包 / 字节 `SpeechEngine` Dialog 实时识别
- iOS：实时语音识别接入，保持现有按住说话 UI 交互

### 2. 情绪发射流程

- 用户输入一句不开心的话后发射
- 火箭发射动画
- 宇宙航线展示
- 到达不同站点的旅程页
- 当前发射内容在站点停留展示

### 3. AI 分析与回复

- 接入豆包大模型分析用户输入
- 返回情绪标签、核心 insight、站点 destination、推荐声景 scene
- AI 吐槽团 5 个角色回复：
  - 嘴替侠
  - 护短朋友
  - 军师
  - 清醒旁观者
  - 降落员
- 当模型失败时，带本地 fallback 兜底逻辑

### 4. 降落模式

- 发射后进入“降落模式”
- 推荐安静方式
- 可播放声音内容
- 支持定时播放
- 默认不自动播放，用户主动点击才播放

### 5. 静心空间

当前静心空间已经升级为三类内容入口：

- 声音
- MV
- 视频

#### 已接入声音

- 微风轻拂
- 海浪拍岸
- 篝火噼啪
- 钟声冥想

#### 已接入 MV

- 你是我放下的那个女人
- 这就是男人

#### 已接入视频

- 海浪
- 篝火
- 河水

### 6. MV / 视频体验

- 列表页真实封面首帧
- 满屏播放页
- 点击画面暂停 / 继续
- 右侧播放按钮暂停 / 继续
- 底部信息卡可展开 / 收起
- 上滑 / 下滑切换同类内容
  - MV 内上下切换 MV
  - 视频内上下切换视频

### 7. 历史记录

- 本地保存发射记录
- 可查看历史输入、分析和落点

## 当前 0.1 版本的边界

这版仍然是偏原型验证 / 单机体验版，主要边界如下：

- 内容仍主要写在客户端代码中
- 没有完整服务端内容管理
- 没有后台上传系统
- 没有把声音 / MV / 视频 / 站点统一做成后台可配置
- 地图站点背景图 / 背景视频还没有接入后台管理

## 后续规划

后续版本会围绕 `dongxin_server` 继续补齐：

- 七牛云上传
- 声音 / MV / 视频后台管理
- 宇宙地图站点管理
- 站点图片 / 视频背景管理
- 自动封面生成
- 自动内容描述生成
- Flutter 客户端从服务端动态拉取内容

## 运行说明

### Flutter 客户端

目录：

- `dongxin_flutter/`

### 服务端

目录：

- `dongxin_server/`

当前仅为骨架初始化，后续会逐步补齐内容后台与七牛集成能力。
